# A Practice Auditing Framework for Large Language Model Use

**Authors:** Yang Zhao, Yingshuo Li, Zeyu Zhang
**Date:** July 3, 2026
**arXiv:** 2607.01248

Proposes a practice auditing framework for LLM use and AI-generated content governance. Introduces "collective empiricism" (LLMs compress human experience into outputs that appear empirical/rational) and "pseudo-rational cognition" (users mistake AI-generated structured expression for their own understanding). Analyzes AI subjectivity illusion, template loops, memory pollution. Proposes auditing via requirement definition, evidence-source auditing, practical validation, reverse questioning, and renewed cognition. Argues LLM outputs should return to verifiable, reproducible, intervenable processes of practice.
